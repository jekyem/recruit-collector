## recruit-collector
  이 프로젝트는 채용 공고를 크롤링 하고, 보여주는 프로젝트 입니다.
    
## 추가 예정
 
  태그 추가 및 검색에 반영

  모바일 대응으로 페이지 수정

  직군 추가
  
  로그인 / 관심설정 / 메일링

  다양한 채용 패이지의 크롤링 모듈 추가

  크롤링 수행 자동화 및 로그 처리

  어드민 페이지 생성
   ( 크롤링 주기 / 로그 / 방문자수 등등 )

페이지 주소 : http://recruit-collector.southeastasia.cloudapp.azure.com/
